Optional: sealed Linux VM

Stronger isolation: each job runs in a disposable Linux guest (Tart). Needs Apple Silicon and ~40 GB free storage. Renters sign into their own Claude/Codex — your Keychain is not used. 

brew install cirruslabs/cli/tart cirruslabs/cli/sshpass
cd /path/to/bay
pnpm --filter @bay/host guest:build
export BAY_SANDBOX_BACKEND=vm
pnpm --filter @bay/host start

Full walkthrough is also on the website under Lend → “Optional: sealed Linux VM”.
